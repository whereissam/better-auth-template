# Implement Feature
1. Read the implementation plan from the referenced file or message
2. Implement all changes across backend and frontend
3. Run build (`bun run build` for TS, `swift build` for iOS)
4. Fix any build errors
5. Run all tests and fix failures
6. Update TODO.md marking completed items with [x]
7. Verify TODO.md was actually written to disk
8. Generate a conventional commit message summarizing changes
9. Stage ONLY the files related to this feature (never `git add .`)
